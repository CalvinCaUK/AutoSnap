# AutoSnap

AutoSnap is a simple Windows screenshot utility for capturing step-by-step workflows using timed or event-triggered screenshots.

It is designed for documenting tutorials, software testing, bug reports, training guides, remote support sessions, VNC sessions, HMI screens, and other repeatable workflows.

## Features

- Timed screenshot capture
- Event-triggered capture from mouse clicks, scroll wheel, and the Enter key
- Multi-monitor support
- Timestamped project folders
- Custom project names and filename prefixes
- Configurable save folder
- Simple Windows GUI
- Can be packaged as a standalone Windows EXE

## Why I built it

Manual screenshots are awkward when documenting a process, testing software, recording a fault, or stepping through a workflow.

AutoSnap automates that process by capturing screenshots either at regular intervals or only when the user interacts with the system.

This can be useful for:

- Creating step-by-step tutorials
- Recording software test evidence
- Capturing bug reports
- Documenting remote support sessions
- Recording VNC workflows
- Capturing HMI or control screen sequences
- Training documentation
- General workflow capture
